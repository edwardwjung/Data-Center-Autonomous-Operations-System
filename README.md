# Data Center Autonomous Operations System

This project demonstrates AI-assisted data center operations across telemetry, risk modeling, optimization, incidenting, and operator visibility.

## Highlights
- Built an end-to-end MVP architecture spanning ingestion, modeling, optimization, alerts, and API/dashboard surfaces.
- Defined module boundaries and shared contracts for cross-service consistency.
- Implemented a runnable simulation + API workflow for iterative development.
- Full operational flow from raw telemetry to ranked recommendations.
- Module decomposition maps to real platform team ownership.
- Shared schema contracts reduce integration ambiguity.
- Recommendation-first safety model supports guarded autonomy.
- Designed for extensibility toward streaming infra and production controls.
