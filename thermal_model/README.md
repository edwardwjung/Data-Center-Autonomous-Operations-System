# Thermal Model Module

This module estimates thermal risk and short-term heat behavior for racks and zones.

## Highlights
- Defined model inputs and forecast horizons for operational use.
- Structured risk outputs for incidenting and optimization consumers.
- Planned evaluation and drift-monitoring checkpoints.
- Generates thermal risk scores from infrastructure signals.
- Produces short-horizon forecasts for proactive operations.
- Serves both incident detection and optimization decisions.
- Includes a roadmap for metrics-driven model governance.
